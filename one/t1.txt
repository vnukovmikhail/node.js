Нет больше белого, нет больше черного,
Есть только серое. Обреченное...
Нет больше радости, нет больше горести,
Только апатия. Зря вы боретесь...

Нет ни агрессии, нет ни оружия,
Слава бездействию, равнодушию.
Нет даже ярости, мир отчуждённости,
Что же останется? Сгустки блёклости.

Улицы города сыпятся красками...
И обесцветились, став ужасными.
Нет больше черного, нет больше белого,
Есть только серое. Омертвелое...
